# End-to-End E-Commerce Data & Sentiment Analysis Pipeline

This repository contains my completed project portfolio fulfilling **Tasks 1 through 4** for my data analytics internship. The project demonstrates a full data pipeline including web scraping, exploratory data analysis, natural language processing (NLP), and data visualization.

##🚀 Project Workflow Breakdown

##🛠️ Task 1: Web Scraping
* Programmatically requested and navigated public e-commerce layout structures using **BeautifulSoup** and **Requests**.
* Collected a custom, structured dataset consisting of product names, prices, and text-based customer review strings.

##🔍 Task 2: Exploratory Data Analysis (EDA)
* Handled data integrity by validating dataset dimensions, identifying duplicate entries, and checking for missing values.
* Cleansed data structures by converting raw string pricing metrics into numeric float formats for computational tracking.

##🧠 Task 4: Sentiment Analysis (NLP)
* Utilized the **NLTK VADER Lexicon engine** to perform text-based sentiment classification on customer review strings.
* Automatically categorized raw customer feedback into distinct psychological brackets: **Positive**, **Neutral**, and **Negative**.

##📊 Task 3: Data Visualization
* Synthesized raw performance arrays into a professional data dashboard using **Matplotlib** and **Seaborn**.
* Visualized structural pricing spreads alongside quantified public opinion trends to support corporate decision-making.

## 📂 Deliverables Included
* **`main_pipeline.ipynb`**: Unified Python source code file containing all project phases.
* **`ecommerce_feedback_data.csv`**: Cleansed tabular custom dataset generated from the live web.
* **`final_internship_dashboard.png`**: High-resolution analytical metrics dashboard showing price distribution and market sentiment.
